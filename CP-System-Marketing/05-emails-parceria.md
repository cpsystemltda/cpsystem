## Como personalizar antes de enviar

**Variáveis a trocar em cada email**:

| Variável | Como preencher |
|---|---|
| `[Nome do sócio]` | Nome completo encontrado no LinkedIn ou site |
| `[Nome]` (remetente) | Quem está enviando (você ou SDR) |
| `[Escritório X]` | Nome do escritório/contabilidade |
| `[ID-personalizado]` | Gerado pelo sistema quando formalizar (ex: `lima-advogados`) |
| `[link YouTube]` | Quando o vídeo estiver pronto |
| `[número WhatsApp]` | Quando você definir |
